# 🐾 PawCare — Pet Health & Marketplace App

A full-featured, single-page web application for managing pet health records and buying or selling pets. Built with HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

**Live demo:** [prestonbc.github.io/pawcare](https://prestonbc.github.io/pawcare)

---

## ✨ Features

### Pet Care Dashboard
- Multi-pet profiles — switch between pets in the sidebar
- Health score indicator per pet
- Upcoming alerts (e.g. flea treatment due, vet bookings)
- Weight trend chart across 6 months

### Health Records
- Full vet visit timeline
- Vaccine tracker with due dates
- Log any health event (checkup, surgery, medication, etc.)

### Medications
- Daily dose scheduler with check-off functionality
- Active prescription list with refill tracking
- Add new medications with dosage and frequency

### Nutrition
- Feeding schedule with morning / midday / evening slots
- Dietary notes and restrictions per pet
- Target weight tracking

### Photo Journal
- Photo gallery grid per pet
- Milestone log (birthdays, moves, firsts)

### 🏪 Marketplace
- Browse listings filtered by species (Dogs, Cats, Rabbits, Birds)
- Search by breed, name, or location
- Full listing detail view — age, sex, vaccination status, seller rating
- Contact seller directly from a listing
- **Sell a pet** — 3-step guided listing form:
  1. Pet details (name, breed, age, sex)
  2. Health & documentation (vaccines, microchip, vet records)
  3. Pricing & seller contact (sale or free adoption)
- **My Listings** — manage active listings, view sold history, seller stats

## 🛠 Tech stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, CSS Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — Fraunces + DM Sans |
| Hosting | GitHub Pages |

No build tools, no npm, no frameworks — opens directly in any browser.
